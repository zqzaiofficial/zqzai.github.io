What is Script API?

What is Script API?

Script API is a toolset for creators to test their new content in Minecraft: Bedrock Edition. It can help you validate that your experiences are running as expected and customize aspects of the Minecraft environment.

Why might we use Script API?

Developers like you and I can use Script API to create custom experiences for players to enjoy. Script API can be used to make gamemodes like Factions, HCF, KitPvP, and more.

How do I get started?

To get started, you will need Minecraft Bedrock Edition, Visual Studio Code and Node.js. You can download them below.
