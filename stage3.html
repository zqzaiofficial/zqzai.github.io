Stage 3/9

Writing our first script!

Let's get to it!

Now that we have our file structure, we can start writing our first script, make sure to open index.js!


First, we need to import the packages we installed in the previous stage. To do this, we need to add the following code to the top of our file:

import { world } from '@minecraft/server';

Now that we have imported our packages, we can start our 'Hello World' script.


In Script API we use the world to register events. For instance if we wanted to run some code when an entity is hurt, we would use the world.afterEvents.entityHurt event.


For this script, we will use the world.afterEvents.chatSend event. This event is fired when a player sends a message in chat.


The script.

// indicates comment.

import { world } from '@minecraft/server';
                
world.afterEvents.chatSend.subscribe(data => {
    if (data.message == "Hello ScriptAPI") {
        var author = data.sender;
        author.sendMessage(`[Script API]: Hello, ${author.name}!`);
    }
});

Explaining the script.

Let's break down the script we just wrote.


First, we import the world from the @minecraft/server package.


Next, we use the world.afterEvents.chatSend.subscribe() function to subscribe to the chatSend event. This function takes a callback function as a parameter, which is called when the event is fired.


Inside the callback function, we check if the message sent is equal to "Hello ScriptAPI". If it is, we get the sender of the message and send them a message.


Now that we have our script, we can run it!
